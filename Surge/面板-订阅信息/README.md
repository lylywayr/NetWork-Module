# 订阅信息

## 说明
这是 `Surge` 的订阅流量信息面板目录，用来显示订阅剩余流量与到期时间。

## 使用方法
- 在 `Surge` 中导入 `Sub-info.sgmodule` 用于单个订阅。
- 在 `Surge` 中导入 `Sub-info_Max.sgmodule` 用于多个订阅。
