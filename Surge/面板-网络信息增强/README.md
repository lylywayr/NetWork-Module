# 网络信息增强

## 说明
这是 `Surge` 的增强网络信息面板目录，用来显示入口出口 IP、ASN、运营商、隐私状态、LAN、SSID 等更详细信息。

## 使用方法
- 在 `Surge` 中导入 `net-lsp-x.sgmodule`。
- 按需调整模块参数，开启或关闭需要显示的字段。
