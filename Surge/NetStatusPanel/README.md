# 网络服务解锁面板（Surge）

模块地址：

```text
https://raw.githubusercontent.com/lylywayr/NetWork-Module/main/Surge/NetStatusPanel/NetStatusPanel.sgmodule
```

在 Surge 的模块页面通过 URL 安装后，策略组页面会显示“网络服务解锁”面板。点击面板刷新按钮可立即重新检测；默认每 300 秒自动更新一次。

检测项目：YouTube、Netflix、Disney+、Spotify、ChatGPT、Claude、Gemini、Grok。

- 🟢：可用且延迟低于 1500 ms
- 🟣：可用但延迟不低于 1500 ms
- 🔴：检测失败或不可用
