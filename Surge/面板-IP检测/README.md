# IP检测

## 说明
这是 `Surge` 的出口 IP 检测面板目录，用来查看当前节点的 IP、地区和运营商信息。

## 使用方法
- 在 `Surge` 中导入 `IP-Check.sgmodule`。
- 刷新面板后查看当前节点详情。
