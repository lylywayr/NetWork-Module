# 轻量流媒体检测

## 说明
这是 `Surge` 的轻量版流媒体解锁检测面板目录，用来快速检测常见流媒体服务的解锁状态。

## 使用方法
- 在 `Surge` 中导入 `Stream-All-Lite.sgmodule`。
- 如需自定义标题、图标或颜色，可通过模块参数调整。
