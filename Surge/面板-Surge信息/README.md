# Surge信息

## 说明
这是 `Surge` 的运行状态面板目录，用来查看运行信息并提供一键重载配置功能。

## 使用方法
- 在 `Surge` 中导入 `SurgePro.sgmodule`。
- 刷新面板后可查看状态，点击可执行配置重载。
