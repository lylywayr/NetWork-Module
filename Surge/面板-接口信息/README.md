# 接口信息

## 说明
这是 `Surge` 的网络接口信息面板目录，用来显示接口速度和流量信息。

## 使用方法
- 在 `Surge` 中导入 `interface-info-normal.sgmodule` 或 `interface-info-ncompact.sgmodule`。
- 根据需要选择常规版或紧凑版样式。
