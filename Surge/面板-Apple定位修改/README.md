# Apple定位修改

## 说明
这是 `Surge` 的 Apple WLOC 定位修改模块目录，用来修改 Apple 网络定位返回坐标，并支持保存、查询和清理自定义位置。

## 使用方法
- 在 `Surge` 中导入 `Apple-WLOC.sgmodule`。
- 通过模块参数填写默认经纬度、精度和日志级别。
- 如需更方便地选点，可使用上游提供的选点页面或快捷指令保存位置。
