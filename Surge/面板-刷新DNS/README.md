# 刷新DNS

## 说明
这是 `Surge` 的 DNS 缓存刷新面板目录，用来快速清理本机 DNS 缓存。

## 使用方法
- 在 `Surge` 中导入 `Flush-DNS.sgmodule`。
- 点击面板即可执行 DNS 缓存刷新。
