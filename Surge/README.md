# Surge

## 说明
这是 `Surge` 客户端模块目录。

当前只保留适合 Surge 使用的面板模块，去广告模块、规则、配置和其它无关内容已经移除。

## 使用方式
- 进入对应中文模块目录。
- 查看该目录下的 `README.md` 了解作用、使用方法和上游来源。
- 选择目录中的 `.sgmodule` 文件作为模块安装入口。

## 维护说明
- 模块目录中的 `script-path` 已统一指向当前仓库对应脚本。
- 每个模块目录的 `README.md` 都保留了上游模块地址和上游脚本地址，便于后续继续拉取更新并重新整理。
