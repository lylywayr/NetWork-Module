# Surge

## 说明
这里是 Surge 客户端模块目录。

本次命名统一按目录前缀区分用途：

- 面板-...：面板展示或信息面板类模块
- 脚本-...：脚本订阅、任务订阅、插件中心适配这类脚本型模块
- 模块-...：普通功能模块

当前仍然遵循你的整理边界：

- 自动化影视仓 不整理到这里
- 自动规则集 不整理到这里
- 去广告仓 不整理到这里

## 使用方式
- 进入对应目录。
- 查看该目录下的 README.md。
- 导入目录中的 .sgmodule 文件。
