# 小米抽奖

## 说明
这是 `小米抽奖.sgmodule` 的整理目录，作为普通功能模块保留。

## 使用方式
- 在 `Surge` 中导入 `小米抽奖.sgmodule`。
- `抽奖开关` 默认留空，表示启用。
- `签到开关` 默认留空，表示启用。
- 将对应参数改为 `#` 可关闭对应脚本。
