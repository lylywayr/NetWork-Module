# 流媒体检测

## 说明
这是 `Surge` 的流媒体解锁检测面板目录，用来检测 Netflix、Disney+、YouTube 等服务的解锁状态。

## 使用方法
- 在 `Surge` 中导入 `Stream-All.sgmodule` 使用完整面板。
- 在 `Surge` 中导入 `Stream-All-Mini.sgmodule` 使用精简面板。
