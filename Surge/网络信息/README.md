# 网络信息

## 说明
这是 `Surge` 的基础网络信息面板目录，用来显示当前出口 IP、地区和运营商信息。

## 使用方法
- 在 `Surge` 中导入 `Network-Info.sgmodule`。
- 刷新面板后查看当前网络信息。
