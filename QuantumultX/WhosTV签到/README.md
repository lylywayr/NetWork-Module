# WhosTV签到

## 说明
这是 `QuantumultX` 的 WhosTV 自动签到模块目录。

## 使用方法
- 在 `QuantumultX` 中导入 `WhosTVSignin.qx.conf`。
- 配置已指向当前目录下的 `WhosTVSignin.js` 原始链接。

## 上游模块地址
- `Configs/QuantumultX/SignIn/WhosTVSignin.qx.conf`

## 上游脚本地址
- `Egern/Modules/WhosTVSignin/WhosTVSignin.js`
- 原外链来源：`https://raw.githubusercontent.com/imzwr214/whos/main/WhosTVSignin.js`
