# 节点重命名

## 说明
这是 `Sub-Store` 的节点重命名脚本目录。

目录内保留脚本本体和说明文档，便于后续继续拉取上游更新并按当前仓库结构维护。

## 使用方法
- 在 `Sub-Store` 的脚本操作中引用 `SubStore-ReName.js`。
- 通过 URL 片段传参，例如 `#name=机场&flag=1&jdqcyg=1`。
- 具体参数说明以脚本头部注释为准。

## 上游模块地址
- `Sub-Store/SubStore-ReName/`
- `Sub-Store/Scripts/`

## 上游脚本地址
- 仓库原始路径：`Sub-Store/SubStore-ReName/SubStore-ReName.js`
- 仓库旧路径：`Sub-Store/Scripts/SubStore-ReName.js`

## 版本说明
- 当前保留的是仓库里较新的 `4.5.2` 版本。
- 旧版本 `4.3` 未再单独保留，避免重复维护。
