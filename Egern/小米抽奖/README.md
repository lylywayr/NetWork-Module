# 小米抽奖

## 说明
这是 `Egern` 的小米抽奖模块目录，用来抓取小米商城抽奖活动 Cookie，并按计划自动执行任务和抽奖。

## 使用方法
- 在 `Egern` 中导入 `MiLottery.yaml`。
- 首次使用时，打开小米商城 App 并进入 `狂欢礼` 或对应抽奖活动页完成抓取。
- 模块内已包含抓取脚本和定时任务，默认每天 `08:30` 执行。
