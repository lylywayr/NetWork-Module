# NetWork-Module

## 说明
这是仓库根目录。

当前仓库按客户端归属整理，只保留模块、面板、小组件和脚本等可复用内容，不再混放不同客户端资源。

- `Sub-Store`
- `Egern`
- `Loon`

## 结构规则
- 一级目录是仓库根目录。
- 二级目录是客户端目录。
- 三级目录是具体模块目录。
- 每一级目录都保留 `README.md` 说明用途和维护方式。
- 模块目录内放同一功能的模块文件、依赖脚本和来源说明。
